# Multimodal-Product-Retrieval-System-
Этот проект реализует мультимодальную систему поиска модной одежды. Основан на датасете Fashion Product Images Dataset с Kaggle.

Возможности:
- Искать товары по тексту (например, «красное летнее платье»)
- Искать по изображению (загрузить фото → найти похожие вещи)
- Гибридный поиск (текст + изображение одновременно)
- Фильтровать по атрибутам: цвет, пол, сезон, тип изделия и др.
